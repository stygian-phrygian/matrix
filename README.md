> All I see is blonde, brunette, redhead.

## Build and Run

First, make sure you have ncurses installed.
On ubuntu: `sudo apt install libncurses-dev`

Then,
```
cmake -S . -B build
cmake --build build
./build/apps/matrix
```
To quit, type `q`
